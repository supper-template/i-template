# ⚡️ 简介

一个开箱即用，基于 `vite 5` + `vue 3` + `typeScript 5` + `pinia` + `vue-router 4` 的PC端项目模板。

# 🚀 开发

1. 安装

```
yarn
```

2. 运行

```
npm run dev
```


# 📦️ 多环境打包

- 测试环境打包

```
npm run build-fix
```

- 生产环境打包

```
npm run build
```

# 🔧 代码检查修复

- 格式检查

```
npm run lint
```

- 自动修复

```
npm run lint-fix
```
